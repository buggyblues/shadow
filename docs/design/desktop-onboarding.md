# OpenClaw Desktop Onboarding 设计方案

## 概述

桌面端 Onboarding 在用户首次打开 OpenClaw 时展示，引导用户：
1. 了解 OpenClaw 是什么
2. 绑定 Buddy（AI 助手）
3. 连接到 Shadow 服务器

---

## 用户流程

```
启动应用 → 欢迎页 → 功能介绍 → Buddy 绑定 → 服务器连接 → 完成
```

## 技术实现

### 文件位置
```
apps/desktop/src/renderer/pages/openclaw/onboard.tsx
```

### 状态管理
```typescript
interface OnboardingState {
  step: 'welcome' | 'features' | 'buddy' | 'server' | 'complete'
  buddyType: 'general' | 'coding' | 'doc' | 'custom'
  apiKey: string
  serverUrl: string
  bindCode: string | null
}
```

### 持久化
- 完成后存储 `openclaw_onboarding_completed: true`
- 下次启动直接进入主界面

---

## 绑定命令

用户也可以通过命令行绑定：

```bash
# 绑定到服务器
openclaw buddy bind --server <server-id>

# 生成绑定码
openclaw buddy code

# 查看状态
openclaw buddy status
```

---

## 与 Web 端配合

1. **Web 端创建服务器后** → 显示绑定引导
2. **用户点击"下载桌面端"** → 下载 OpenClaw
3. **桌面端打开** → 进入 Onboarding
4. **完成绑定** → 返回 Web 端，Buddy 已就绪
