# 虾豆 V1.0 运营冷启动产品方案

> **版本**: V1.0  
> **目标**: 完成租赁市场核心链路 + Buddy需求发布 + 官方Claw冷启动  
> **状态**: 设计阶段  
> **创建时间**: 2026-03-19

---

## 1. 产品愿景

虾豆是一个 **Claw（AI Agent）P2P 租赁市场 + 能力互补平台**。

**核心价值主张:**
- **对Claw所有者**: 闲置算力变现，让Claw为你赚钱
- **对Claw租赁者**: 低成本获取专业AI能力，无需自己搭建
- **对需求发布者**: 零技术门槛发布需求，找到合适的Claw完成
- **对Claw开发者**: 能力变现，帮助非技术人员解决问题

#### 3.1.3 关键功能点

| **违约处理** | P1 | 违规使用检测与押金扣除 |

#### 3.1.4 技术实现要点

**归属权切换机制:**
```typescript
// 租赁期间Claw归属权模型
interface ClawOwnership {
  clawId: string;
  ownerId: string;        // 原始所有者
  tenantId: string | null; // 当前租户（租赁期间）
  status: 'idle' | 'rented' | 'maintenance';
  currentContractId: string | null;
  rentedAt: Date | null;
  expiresAt: Date | null;
}

// 权限检查逻辑
function canUseClaw(clawId: string, userId: string): boolean {
  const ownership = getClawOwnership(clawId);
  if (ownership.status === 'rented') {
    return ownership.tenantId === userId; // 只有租户可用
  }
  return ownership.ownerId === userId; // 空闲时所有者可用
}
```

**虾币支付流程:**
```
1. 租户发起租赁请求
2. 系统计算预估费用（押金 + 预付费）
3. 租户虾币余额检查
4. 锁定押金 + 预付费用
5. 租赁期间按实际使用计费
6. 租赁结束结算，退还剩余押金
```

### 3.3 模块3: 官方Claw冷启动

#### 3.3.1 冷启动策略

**核心思路:**
> 在P2P市场成熟前，先用官方Claw验证核心玩法，建立用户认知，积累种子用户。

**官方Claw定位:**
- **质量标杆**: 展示虾豆Claw的能力上限
- **玩法验证**: 测试哪些场景用户真正愿意付费
- **种子培养**: 培养第一批核心用户

#### 3.3.2 官方Claw场景规划

| Claw名称 | 场景 | 目标用户 | 收费模式 |
|----------|------|----------|----------|
| **Excel助手** | 数据处理、公式、图表 | 办公人群 | 按任务收费 |
| **文案生成器** | 小红书/朋友圈/邮件文案 | 自媒体/运营 | 按条收费 |
| **代码小助手** | 脚本编写、Bug修复、代码解释 | 初学者/非技术 | 按时长收费 |
| **文件整理员** | 批量重命名、分类、格式转换 | 所有人 | 按任务收费 |
| **数据分析师** | CSV分析、简单可视化、报告生成 | 小商家/个人 | 按任务收费 |

#### 3.3.3 运营策略

**第一阶段（前100用户）:**
- 免费体验，收集反馈
- 1对1用户访谈
- 快速迭代场景

**第二阶段（100-1000用户）:**
- 引入虾币计费
- 推荐奖励机制
- 用户UGC内容激励

**第三阶段（1000+用户）:**
- 开放P2P市场
- 官方Claw转为"示范店"
- 扶持优质第三方Claw

## 4. 虾币体系设计

### 4.1 虾币基础规则

| 属性 | 设定 |
|------|------|
| **名称** | 虾币 (XiaBi / XB) |
| **汇率** | 1 CNY = 100 虾币 |
| **最小单位** | 1 虾币 (0.01 CNY) |
| **平台抽成** | 5% (租赁交易) |

### 4.2 虾币流动

```
【虾币生态系统】

        充值                    租赁支付
    ┌──────────┐            ┌──────────┐
    │          │            │          │
    ▼          │            ▼          │
┌───────┐      │        ┌───────┐      │
│ 用户  │──────┘        │ 租户  │──────┘
│ 钱包  │◀──────────────│ 钱包  │
└───────┘   租赁收入     └───────┘
    │
    │ 提现
    ▼
┌───────┐
│ 银行  │
│ 账户  │
└───────┘

【平台收入】
- 租赁交易抽成: 5%
- 提现手续费: 1% (最低1元)
```

### 4.3 钱包功能

| 功能 | 说明 |
|------|------|
| **充值** | 微信支付、支付宝 |
| **余额查询** | 可用余额、冻结余额 |
| **交易记录** | 充值、消费、收入、提现明细 |
| **提现** | 绑定银行卡，T+1到账 |
| **自动扣款** | 租赁开始时预扣，结束时结算 |
