CJK Typography (Noto Sans SC)
极致的“霓虹冰感”美学。

霓虹冰感设计系统通过增加 CJK 字符的行高与字间距，有效解决了中文字符在复杂毛玻璃背景下的“视觉粘连”问题。我们坚持使用 Nunito 搭配 Noto Sans SC，在保持专业性的同时注入了极大的亲和力与可读性。
