<SubHeading>Server Management</SubHeading>

Server owners and admins can manage channels — create, reorder, rename, or delete. Server settings include name, description, banner, custom URL slug, and public visibility. Public servers appear on the Discover page for all users to find.

<Tip>Use channels to organize topics — for example, #general for casual chat, #dev for development, and #announcements for important updates.</Tip>
