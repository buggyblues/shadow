# 首页玩法

Shadow 首页玩法面向“点击即用”的新用户路径。新用户选择一个玩法后，可以用邮箱验证码或 OAuth
快速登录，看到一段简洁的准备过程，然后直接进入正确的频道或 Cloud 部署出来的 Buddy 空间。

## 点击玩法后会发生什么

<Step num={1} title="选择玩法">从首页选择公开社区玩法、私人频道玩法，或 Cloud 模板玩法。</Step>
<Step num={2} title="快速登录">使用邮箱验证码、Google 或 GitHub 登录。注册和体验社区功能不再要求邀请码。</Step>
<Step num={3} title="准备空间">Shadow 会执行配置好的动作：加入已有服务器、创建你的私人频道，或部署 Cloud 模板。</Step>
<Step num={4} title="进入频道">空间准备好后，Shadow 会打开默认频道，Buddy 会在那里向你打招呼。</Step>

## Cloud 玩法

Cloud 玩法不会创建假的空服务器，而是运行 Shadow Cloud 目录里的真实模板。部署页面可以选择：

- **官方供应商**：使用 Shadow 官方 OpenAI-compatible 模型代理，并从你的虾币钱包计费。
- **我的供应商**：使用你自己加密保存的供应商配置，模型费用由你和该供应商直接结算。

官方上游 key 只保存在 Shadow 服务端，不会注入到你的服务器或 Buddy 运行环境里。

<Tip>玩法完成后会进入配置的默认频道。如果服务器暂时没有频道，应用会展示空状态，而不是跳到无法继续的路由。</Tip>
